What is Full Stack Development?

Full Stack Development refers to the development of both front-end (client side) and back-end (server side) portions of web applications.

A full stack developer is capable of working on all layers of a software application, from the user interface to the server, database, and everything in between.

HTML

HTML (HyperText Markup Language) is the standard markup language for documents designed to be displayed in a web browser.

It provides the structure of a webpage and can be enhanced and modified by other technologies like CSS and JavaScript.

HTML5 Semantic Elements
  • <header> - Defines a header for a document or section
  • <nav> - Defines navigation links
  • <section> - Defines a section in a document
  • <article> - Defines independent, self-contained content
  • <footer> - Defines a footer for a document or section
HTML Code Example
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>My Webpage</title>
</head>
<body>
    <header>
        <h1>Website Title</h1>
        <nav>
            <ul>
                <li><a href="#home">Home</a></li>
                <li><a href="#about">About</a></li>
            </ul>
        </nav>
    </header>
   
    <main>
        <article>
            <h2>Article Title</h2>
            <p>Article content...</p>
        </article>
    </main>
   
    <footer>
        <p>Copyright Info</p>
    </footer>
</body>
</html>

Bootstrap Framework
